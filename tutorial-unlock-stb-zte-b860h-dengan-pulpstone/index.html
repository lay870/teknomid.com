Selengkapnya, langkah-langkahnya sebagai berikut :
  • Laptop atau komputer dinyalakan, kemudian jalankan tool flasher setelah sebelumnya kita ekstrak file hasil download , cari dan klik Pulpstone_ZTE_B860H_Flasher_Tool_v3.1.exe

Registrasi Untuk Menggunakan Firmware Pulptone Android v3.1

Setelah berhasil melakukan unlock atau flash STB ZTE ZXV10 B860H dengan firmware Pulpstone Android v3.1, maka selanjutnya kita harus registrasi, dikenai biaya donasi saja murah kok saya cuma isi pulsa saja ke Pak Dokter untuk perawatan server dan pengembangan firmware tentunya.   • Buka terminal yang sudah terinstal di Pulpstone Android ini untuk melakukan registrasi
